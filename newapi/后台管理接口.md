# Authentication

* API Key (SessionAuth1)
    - Parameter Name: **session**, in: cookie. Session认证，通过登录接口获取

* API Key (AccessToken1)
    - Parameter Name: **Authorization**, in: header. Access Token认证，格式: Bearer {access_token}，通过 /api/user/token 接口生成

* API Key (NewApiUser1)
    - Parameter Name: **New-Api-User**, in: header. 用户ID请求头，必须与当前登录用户ID匹配，使用Session或AccessToken认证时必须提供
